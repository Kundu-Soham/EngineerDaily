# Engineer Daily

A simple, production-ready web application that generates concise daily AI-powered news briefings for engineering professionals.

## Architecture
- **Backend:** FastAPI, Python 3.12+, OpenAI API, Pydantic
- **Frontend:** Next.js (App Router), React, Tailwind CSS

## Prerequisites
- Python 3.12+
- Node.js 18+
- OpenAI API Key
